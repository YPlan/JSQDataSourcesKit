TableViewFetchedResultsDelegateProvider

public final class TableViewFetchedResultsDelegateProvider <CellFactory: CellFactoryProtocol where CellFactory.Cell: UITableViewCell>: CustomStringConvertible

A TableViewFetchedResultsDelegateProvider is responsible for providing a delegate object for an instance of NSFetchedResultsController that manages data to display in a table view.

Warning

The CellFactory.Item type should correspond to the type of objects that the NSFetchedResultsController fetches.
