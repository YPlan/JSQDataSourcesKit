CollectionSupplementaryViewFactory

public struct CollectionSupplementaryViewFactory <SupplementaryView: UICollectionReusableView, Item>: CollectionSupplementaryViewFactoryType, CustomStringConvertible

A CollectionSupplementaryViewFactory is a concrete CollectionSupplementaryViewFactoryType. This factory is responsible for producing and configuring supplementary views for a collection view for a specific item.

Properties
  • reuseIdentifier

    A unique identifier that describes the purpose of the supplementary views that the factory produces. The factory dequeues supplementary views from the collection view with this reuse identifier.

    Note

    Clients are responsible for registering a view for this identifier and a supplementary view kind with the collection view.
