TableViewCellFactory

public struct TableViewCellFactory <Cell: UITableViewCell, Item>: TableViewCellFactoryType, CustomStringConvertible

A TableViewCellFactory is a concrete TableViewCellFactoryType. This factory is responsible for producing and configuring table view cells for a specific item.

Properties
  • reuseIdentifier

    A unique identifier that describes the purpose of the cells that the factory produces. The factory dequeues cells from the table view with this reuse identifier.

    Note

    Clients are responsible for registering a cell for this identifier with the table view.
